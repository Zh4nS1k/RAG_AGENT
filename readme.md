# Юридический RAG-агент по законодательству РК

Система поиска и ответов на вопросы на основе Кодексов Республики Казахстан с использованием технологий RAG (Retrieval-Augmented Generation).

## Особенности
- **Источники данных:** Официальный сайт ИПС «Әділет» (adilet.zan.kz).
- **LLM:** Google Lyria 3 Pro Preview (через OpenRouter).
- **Эмбеддинги:** `intfloat/multilingual-e5-small` (поддержка русского и казахского языков).
- **Хранилище:** Dual-storage — PostgreSQL (метаданные и полные тексты) + ChromaDB (векторный поиск).

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   make install
   ```
3. Настройте API ключ в файле `.env`:
   ```env
   OPENROUTER_API="ваш_ключ_здесь"
   ```

## Использование
1. Запустите базу данных:
   ```bash
   make db-up
   ```
2. Проведите индексацию (загрузку законов):
   ```bash
   make index
   ```
3. Запустите агента для ответов на вопросы:
   ```bash
   make run
   ```

## Структура проекта
- `main.py` - основной цикл работы агента и генерация ответов.
- `rag_data.py` - скрипт парсинга сайта adilet.zan.kz и индексации в БД.
- `database.py` - описание моделей SQLAlchemy для PostgreSQL.
- `api.py` - конфигурация API ключей.
