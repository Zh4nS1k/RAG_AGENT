# Инструкции для Claude / LLM по работе с проектом

## Контекст
Этот проект — RAG система для работы с законодательством Казахстана.

## Правила парсинга (adilet.zan.kz)
- Используйте теги `<h3>` как разделители статей.
- Сохраняйте `id` статьи в метаданных для формирования прямых ссылок (anchor links).
- Тексты статей находятся в тегах `<p>`.

## Эмбеддинги
- Всегда используйте `multilingual-e5-small` для обеспечения поддержки кириллицы и специфичных терминов.

## База данных
- PostgreSQL используется как Master-система для хранения исходного текста.
- ChromaDB используется только для быстрого векторного поиска по чанкам.
- При получении чанка из Chroma, желательно подтягивать контекст всей статьи из PostgreSQL через `postgres_article_id`.
